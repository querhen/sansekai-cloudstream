
# Sansekai Cloudstream Repository

Professional Cloudstream extension repository for short drama providers.

## Install

Add this repo inside Cloudstream:

https://raw.githubusercontent.com/querhen/sansekai-cloudstream/main/repo.json

## Contents

Plugin:
- SansekaiDrama

## Features

- Trending dramas
- Latest updates
- Search
- Episode listing
- Stream extraction ready

## Build

This repository uses GitHub Actions to automatically compile the plugin (.cs3).

Steps:
1. Upload this repository to GitHub
2. GitHub Actions will run automatically
3. Download the generated `.cs3` plugin artifact
4. Upload it to `/plugins/SansekaiDrama.cs3`
