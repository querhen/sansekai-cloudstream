## 📺 Tentang Extension
Repository ini menyediakan plugin **Sansekai** untuk aplikasi **Cloudstream**. Dibuat khusus untuk mengakses konten dari API Sansekai secara gratis tanpa login dan tanpa iklan.

### ✨ Fitur Utama
* **Dracin & DramaBox:** Akses drama China terbaru.
* **ShortMax & ReelShort:** Koleksi drama pendek (vertical drama).
* **Anime & Komik:** Update konten harian.
* **High Quality API:** Menggunakan server VPS yang sudah di-upgrade oleh tim Sansekai.

---

## 📥 Cara Instalasi

Pilih salah satu metode di bawah ini untuk menambahkan repository ini ke aplikasi Cloudstream kamu:

### ⚡ Metode 1: Klik Otomatis
Jika kamu membaca ini di perangkat yang sudah terinstall Cloudstream, cukup klik tombol di bawah:

[<img src="https://img.shields.io/badge/Add%20to-Cloudstream-blueviolet?style=for-the-badge&logo=android&logoColor=white">](cloudstream-repo://https://raw.githubusercontent.com/querhen/sansekai-cloudstream/gh-pages/plugins.json)

### 🛠️ Metode 2: Manual Link
1.  Buka aplikasi **Cloudstream**.
2.  Pergi ke **Settings** > **Extensions**.
3.  Klik **Add Repository**.
4.  Masukkan nama: `Sansekai`
5.  Masukkan URL berikut:
    ```text
    [https://raw.githubusercontent.com/querhen/sansekai-cloudstream/gh-pages/plugins.json](https://raw.githubusercontent.com/querhen/sansekai-cloudstream/gh-pages/plugins.json)
    ```
